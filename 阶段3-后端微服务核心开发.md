# 阶段3：后端微服务核心开发（第5-8周）

## 阶段目标

- Spring Cloud父工程搭建完成
- Gateway网关服务开发完成
- 七大微服务全部开发完成
- 所有微服务本地联调通过
- 四库异构数据协同流转实现
- 后端项目Maven打包成功

## 阶段产出

1. Spring Cloud父工程及所有子模块
2. Gateway网关服务（路由/鉴权/限流/跨域）
3. 用户权限服务（注册/登录/RBAC）
4. 设备管理服务（CRUD/状态监测）
5. 数据采集服务（接收/校验/过滤/预处理）
6. 数据存储服务（四库统一写入/查询）
7. 缓存调度服务（Redis工具/分布式锁/限流）
8. 异常预警服务（阈值比对/预警触发）
9. AI调用服务（Python接口桥接）
10. Maven打包Jar文件
