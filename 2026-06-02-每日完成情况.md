```java
public Map<String, Object> login(String username, String password) {
    // 1. 查询用户
    User user = this.getOne(wrapper);
    
    // 2. 验证密码（BCrypt）
    if (!passwordEncoder.matches(password, user.getPassword())) {
        throw new BusinessException("密码错误");
    }
    
    // 3. 生成JWT Token
    String token = jwtUtil.generateToken(user.getId(), username, claims);
    
    // 4. 存储到Redis（24小时过期）

    return result;
}
```

**关键代码（FourDatabaseStorageService.java）：**
```java
public Map<String, Object> storeToAllDatabases(HealthRecord record) {
    Map<String, Object> results = new HashMap<>();
    
    try {
        storeToMySQL(record);      // 关系型数据
        results.put("mysql", true);
    } catch (Exception e) {
        results.put("mysql", false);
    }
    
    try {
        storeToRedis(record);      // 缓存层
        results.put("redis", true);
    } catch (Exception e) {
        results.put("redis", false);
    }
    
    try {
        storeToMongoDB(record);    // 文档型数据
        results.put("mongodb", true);
    } catch (Exception e) {
        results.put("mongodb", false);
    }
    
    return results;
}
```

**Redis存储策略：**
- 最新数据Key: `storage:latest:{userId}:{dataType}` （Hash结构，7天TTL）
- 历史数据Key: `storage:history:{userId}:{deviceId}` （List结构，保留最近1000条，30天TTL）

#### 12.1 异常预警服务 zhihealth-alert（100%完成）🆕

| 文件 | 功能 |
|------|------|
| AlertApplication.java | 启动类（@SpringBootApplication + @EnableDiscoveryClient） |

**AlertEngineService核心功能：**
```java
public Map<String, Object> evaluateData(Map<String, Object> healthData) {
    // 1. 根据数据类型获取匹配的预警规则
    List<AlertRule> rules = alertRuleService.getRulesByDataType(dataType);
    
    // 2. 逐条规则评估
    for (AlertRule rule : rules) {
        boolean triggered = evaluateRule(rule, healthData);  // gt/gte/lt/lte/eq/neq
        
        // 3. 冷却机制检查
        if (triggered && !isCooldownActive(rule.getId())) {
            Map<String, Object> alertInfo = createAlertRecord(rule, healthData);
            saveAlertAsync(rule, healthData, alertInfo);  // 异步保存
            triggeredAlerts.add(alertInfo);
        }
    }
    
    return result;  // {evaluated, alerts[], totalRules, triggeredCount}
```

```java
@Component
public class RateLimiterFilter implements GlobalFilter, Ordered {
    // 基于Redis的滑动窗口限流算法
    
    private boolean isRateLimited(String clientIp, String path) {
        String key = "rate_limit:" + clientIp + ":" + path;
        
        Long currentCount = redisTemplate.opsForValue().increment(key);
        if (currentCount == 1) {
            redisTemplate.expire(key, Duration.ofMinutes(1));  // 1分钟窗口
        }
        
        int limit = getRateLimitForPath(path);  // 按路径动态限流
        
        return currentCount > limit;  // 超限返回true
    }
}
```

**限流策略（按API路径）：**
